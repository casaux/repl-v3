This application form is to be used to make an application for a remote pilot licence (RePL) in accordance with regulation 101.290 of the Civil Aviation Safety Regulations 1998.

Persons listed on this application will be checked against CASA records for existing RePLs. Any person with an existing RePL will be issued a variaton to their existing RePL while any person with no existing RePL will receive an initial issue if they meet the eligibility requirement.
